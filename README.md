# Preston Cosby — Personal Portfolio

The personal portfolio of **Preston Cosby** — biochemistry researcher and Student Body President at NCSSM, building toward cost-effective, equitable healthcare at the intersection of bioengineering and public policy.

🔗 **Live:** https://prestoncosby.com

---

## About

A single-page portfolio site covering research, real-world impact, interests, and résumé — designed to be the first thing attached to an email and the last thing someone forgets after seeing it.

## Stack

- Self-contained static site — HTML, CSS, and vanilla JavaScript in a single file
- Typography: Fraunces, Hanken Grotesk, JetBrains Mono
- Hosted on **Netlify**; contact form handled via Netlify Forms
